Solar Tracking System

Arduino Uno & LDR-based solar panel alignment for maximum energy efficiency

Project by Koushik Gosh
Overview

This project uses two LDR sensors and an Arduino Uno to keep a solar panel aligned with the sun’s position throughout the day, improving its energy capture efficiency compared to static panels.

Components Required

  • Arduino Uno (x1)
  • LDR sensors (x2)
  • 10kΩ resistors (x2)
  • Servo Motor (e.g., SG90/MG996R) (x1)
  • Breadboard (x1)
  • Jumper wires (as needed)
  • Solar panel (optional) (x1)
  • Cardboard/plastic mount (x1)
  • Power supply (USB or external 5V) (x1)

How It Works

The two LDRs detect light intensity from left and right. If one side detects more light, the servo rotates the panel toward that side. If the light levels are balanced, the panel holds position. This keeps the panel facing the sun directly.

Analog A0

Analog A1

Arduino Code (Excerpt)

#include <Servo.h>

Servo solarServo;
const int ldrLeft = A0;
const int ldrRight = A1;
int pos = 90; // initial position

void setup() {
  solarServo.attach(9);
  solarServo.write(pos);
  Serial.begin(9600);
}

void loop() {
  int leftValue = analogRead(ldrLeft);
  int rightValue = analogRead(ldrRight);
  int threshold = 50;

  if (abs(leftValue - rightValue) <= threshold) return;

  if (leftValue > rightValue) pos++;
  else pos--;

  pos = constrain(pos, 0, 180);
  solarServo.write(pos);
  delay(100);
}

Testing & Calibration

  • Test indoors and outdoors under different light conditions.
  • Adjust threshold in code for your LDR sensitivity.
  • Ensure servo doesn’t jitter by fine-tuning delay and threshold.
