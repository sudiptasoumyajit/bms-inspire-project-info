Wi-Fi Controlled Car

ESP8266-based robot car with web interface for real-time movement control

Project by Ritam Ghosh
Overview

This project uses an ESP8266 (NodeMCU) to create a Wi-Fi-enabled robot car controllable via a simple web page hosted directly on the microcontroller. It uses an L298N motor driver to operate two DC motors for movement in all directions.

Hardware Components

  • ESP8266 (NodeMCU) — Wi-Fi microcontroller & web server
  • L298N Motor Driver — Controls 2 DC motors with H-Bridge
  • 2× DC Motors + Wheels — Movement
  • Power Supply — 7.4V–12V for motors, 5V for ESP
  • Chassis — Mount for electronics and motors
  • Jumper Wires — Connections
  • Optional: Ultrasonic sensor — Obstacle avoidance

Software & Programming

  • Programmed using Arduino IDE with ESP8266WiFi.h and ESP8266WebServer.h.
  • Two modes:
    • AP Mode: Phone connects directly to car’s Wi-Fi.
    • STA Mode: Car connects to your home Wi-Fi network.
  • Web server hosts control page with Forward, Backward, Left, Right, Stop buttons.
  • Button press sends HTTP GET request (e.g., /forward).
  • ESP8266 interprets request and sets motor driver pins accordingly.
// Example: Move Forward
if (server.uri() == "/forward") {
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, HIGH);
  digitalWrite(IN4, LOW);
}

Communication Flow

  • User clicks control button on webpage.
  • Browser sends HTTP request to ESP8266.
  • ESP8266 decodes request and sets GPIO outputs.
  • Motor driver spins motors in correct direction.

Enhancements

  • Replace HTTP with WebSockets for instant, low-latency control.
  • Add PWM speed control.
  • Integrate camera module (ESP32-CAM) for live FPV streaming.
  • Add ultrasonic sensors for autonomous obstacle avoidance.
  • Battery voltage monitoring.

Troubleshooting Tips

  • No power to ESP8266 — Check 5V VIN pin & GND wiring.
  • Motors not moving — Verify IN1–IN4 connections & logic signals.
  • Response delay — Optimize code or use WebSockets.
  • ESP resets — Voltage drop; add capacitors or separate ESP power.
