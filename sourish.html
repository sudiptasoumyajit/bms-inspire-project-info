Smart Parking System

Designed and Developed by Sourish Koner

Project Overview

The Smart Parking System is an Arduino-based solution for efficient vehicle parking management. Using ultrasonic or IR sensors, it detects the availability of parking slots and provides real-time status updates to users.

Mechanism
  • Sensors Detection: Ultrasonic or IR sensors at each parking slot detect presence by measuring distance or beam obstruction.
  • Arduino Microcontroller: Collects and processes data from sensors to determine slot availability.
  • Display System: Shows status of each slot on LCD/LED display (e.g., "Slot 1: Free") and optionally on mobile/web via Wi-Fi (ESP8266) or Bluetooth.
  • Entry Control (Optional): Servo motor or barrier gate allows/restricts entry based on availability.
  • Power Supply: Operates on DC source or batteries.
Benefits
  • Reduces time spent searching for parking.
  • Provides real-time monitoring.
  • Scalable for large parking lots.
Quick Facts

Controller: Arduino

Sensors: Ultrasonic / IR

Connectivity: Wi-Fi (ESP8266) / Bluetooth

Display: LCD / LED

Power: DC / Battery

Possible Upgrades
  • Integrate mobile app notifications
  • Automated payment system
  • License plate recognition
